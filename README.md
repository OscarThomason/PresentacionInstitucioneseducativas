# Presentacion Instituciones Educativas

Sitio estatico listo para publicarse en GitHub Pages.

## Publicacion en GitHub Pages

1. Sube estos cambios a la rama `main`.
2. En GitHub, abre `Settings > Pages`.
3. En **Build and deployment**, selecciona **Source: GitHub Actions**.
4. Espera a que termine el workflow **Deploy static site to GitHub Pages**.
5. Tu sitio quedara disponible en:
   - `https://oscarthomason.github.io/PresentacionInstitucioneseducativas/`

## Desarrollo local

Abre `index.html` en el navegador o levanta un servidor estatico simple.
